### 2.浏览器问题

![image-20210314161417921](D:\web\抖音项目\cou-yin\README.assets\image-20210314161417921.png)

#### 修改：在postcss.config.js中

```
        'autoprefixer':{
            overrideBrowserslist: [
              "Android 4.1",
              "iOS 7.1",
              "Chrome > 31",
              "ff > 31",
              "ie >= 8"
              //'last 10 versions', // 所有主流浏览器最近10版本用
          ],
          grid: true
        },
```
